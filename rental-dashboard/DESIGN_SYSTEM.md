# Galant Design System

## Color Palette

### Brand Colors
- **Primary (Emerald Green)**: `#10b981` - Active states, branding, CTAs
- **Accent (Bright Orange)**: `#fb923c` - Highlights, submenu active states, warnings

### Semantic Colors
- **Blue-500**: `#3b82f6` - Information, links
- **Orange-500**: `#f97316` - Alerts, important notifications  
- **Indigo**: `#6366f1` - Secondary actions, badges
- **Destructive (Red)**: `#ef4444` - Errors, delete actions

### Sidebar (Dark Theme)
- **Background**: `#121212` (near-black)
- **Text**: Slate-100 (#f1f5f9)
- **Border**: Dark gray (#333333)

### Active/Selected States
- **Nav Active**: Primary with 15% opacity (`bg-primary/15`)
- **Submenu Active**: Accent with 15% opacity (`bg-accent/15`)

## Typography
- **Brand Font**: Permanent Marker (cursive, handwritten)
- **Body Font**: Figtree (sans-serif)

## Usage Guidelines
- Use **Primary** for main navigation and key actions
- Use **Accent** for secondary highlights and submenu selections
- Dark sidebar provides professional contrast with light content area
- Transparent backgrounds (15% opacity) for subtle selection indicators
